Generelt

På skolen jobber vi med ulike prosjekter innen programmering, webutvikling og medieproduksjon. Dette inkluderer utvikling av nettsider, produksjon av digitale medier som videoer, og redigering av bilder. Vi lærer om flere forskjellige kodespråk, og så langt har vi fått en god innføring i HTML, CSS og JavaScript. HTML er det grunnleggende språket som blir brukt til å strukturere innholdet på nettsidene våre, slik som overskrifter, avsnitt og bilder. CSS lar meg kontrollere utseendet til nettsider, slik at vi kan tilpasse farger, fonter, layout og andre visuelle elementer for å gjøre sidene mer attraktive og brukervennlige.

Medieproduksjon

I medieproduksjon lagde jeg en kort julefilm. Det var en bra mulighet til å jobbe med alt fra planlegging til filming og redigering. Jeg lærte mye om hvordan man kan bruke video til å fortelle en historie og gjøre det spennende å se på. Jeg lærte også mye om hvordan kamera fungerer og hvordan de ulike innstillingene på kameraet brukes.

Engelsk

Jeg har laget en engelsk nettside om turisme. Det var gøy å kombinere design og funksjonalitet for å lage noe som både ser bra ut og fungerer skikkelig. Jeg brukte HTML og CSS for å bygge opp siden, og JavaScript for å legge til kule funksjoner som gjør siden mer spennende å bruke.

JavaScript

Når vi jobber med JavaScript, gjør vi oppgaver for å lære hvordan det fungerer. For eksempel har jeg laget en funksjon som teller små bokstaver i en tekst. Det hjalp meg å skjønne hvordan koding kan brukes til å løse oppgaver og gjøre ting enklere på nettsider. Det er gøy å se hvordan det jeg lager faktisk fungerer i praksis
